### Metronome

A Web-App of a metronome, built with Svelte and Vite. It allows users to set a tempo in beats per minute (BPM) and provides visual and auditory cues to help musicians keep time while practicing or performing.

#### Stacks

- TypeScript
- Vite
- Svelte
- Tailwind CSS

#### Requirements

- A metronome that allows users to set a tempo in beats per minute (BPM) and provides visual and auditory cues.
- The metronome should have a user-friendly interface that allows users to easily adjust the tempo and start/stop the metronome.
- The metronome should be responsive and work well on both desktop and mobile devices.
- The metronome should have a visual representation of the beats, such as a flashing light or a moving pendulum, to help users keep time.
- The metronome should support presets for common tempos, such as 60 BPM, 120 BPM, and 180 BPM, and allow users to save their own presets for easy access.
- The preset should be stored in the local storage of the browser, so that they persist across sessions.
- The metronome should play a different sound for the first beat of each measure to help users keep track of the rhythm.
